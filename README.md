# C Language Lecture Video Course Schedule

This repository contains a structured 10-lecture video course designed to teach the fundamentals of C Programming. Each lecture includes clear topics, outcomes, and assignments for practice.

---

## **Lecture 1: Introduction to C Programming (10 mins)**  
### **Topics**
- What is C? Features & Applications
- Structure of a C program
- Writing your first C program
- Compilation and Execution Process

### **Outcome**
Learners understand the basics and can write, compile, and execute a simple program.

### **Assignment**
Write a **"Hello, World!"** program and print your name, age, and course.

---

## **Lecture 2: Variables, Data Types & Operators (10-15 mins)**
### **Topics**
- Variable types
- Integer, float, char, double
- Arithmetic, relational, logical, assignment operators
- Increment/decrement

### **Outcome**
Able to use different data types and operators in expressions.

### **Assignment**
Calculate **area**, **perimeter**, **simple interest**, and use **++/--** in expressions.

---

## **Lecture 3: Input/Output Functions & Format Specifiers (10-15 mins)**
### **Topics**
- printf(), scanf()
- %d, %f, %c, %s, %lf
- Escape sequences
- Type conversion & type casting

### **Outcome**
Student can handle user input and output clearly.

### **Assignment**
Take **5 numbers** from the user and display formatted output.

---

## **Lecture 4: Decision-Making Statements (10-15 mins)**
### **Topics**
- if, if-else, nested-if
- Ladder if
- Logical decisions
- Real-life examples

### **Outcome**
Able to implement decision-making in programs.

### **Assignment**
Check **positive/negative**, **even/odd**, **largest of three numbers**.

---

## **Lecture 5: Switch Case & Conditional Operator (10-15 mins)**
### **Topics**
- switch syntax and rules
- Menu-driven programs
- Ternary operator examples

### **Outcome**
Efficient use of switch for multiple conditions.

### **Assignment**
Create a **calculator** using switch.

## **Lecture 7: Pattern Printing (20 mins)**
### **Topics**
- Star patterns
- Number patterns
- Single loop logic vs nested loop
- Pyramid, triangular patterns

### **Outcome**

### **Assignment**

---

## **Lecture 8: Arrays & Strings (10-15 mins)**
### **Topics**
- 1D, 2D arrays
- Array operations (max, min, sum, average)
- Strings: gets(), fgets(), puts(), strlen(), strcpy(), strcmp()

### **Outcome**
Understanding of array storage and string handling.

### **Assignment**
- Store **10 numbers**, find **max/min**.
- Accept a **string** and count **vowels**.

---

## **Lecture 9: Functions & Recursion (10-15 mins)**
### **Topics**
- Need of functions
- Call by value, call by reference
- Recursive functions
- Problems on recursion

### **Outcome**
Able to modularize code with user-defined functions.

### **Assignment**
Write recursive programs for **factorial**, **Fibonacci**, and **sum of numbers**.
